# DaruDaru

마루마루 만화 다운로더

![discord](img/discord.jpg)

[Download](https://github.com/RyuaNerin/DaruDaru/releases/latest)

## 특징

- zip 으로 압축해줌

- 고속 다운로드

## 사용법

- 마루마루 링크를 주면 해당 만화 전체 다운로드

- 만화 링크를 주면 해당 만화만 다운로드

- 검색, 다운로드 기록 저장

- 다운로드 폴더 안에 마루마루 링크 생성
